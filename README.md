# 🎮 Pixel Ledger

## 📌 Project Overview

Pixel Ledger is a modern web-based platform that enables users to explore, discover, and analyze video games using real-time data from an external API. The application provides a visually rich and immersive interface for browsing games, viewing detailed information, and navigating across different categories.

---

## 🎯 Purpose

The purpose of this project is to build a dynamic web application that demonstrates the ability to fetch, process, and present external data in an intuitive and structured format. It focuses on creating a seamless browsing experience where users can efficiently explore large datasets through interactive UI components.

---

## 🌐 API Used

This project uses the **RAWG Video Games Database API**.

Features of the API:

* Access to a large collection of video games
* Game ratings and metadata
* Genres and platforms
* Images and screenshots

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/thedittya471/Pixel-Ledger-Wap-Project.git
```

2. Navigate to the project folder:

```
cd Pixel-Ledger-Wap-Project
```

3. Open `index.html` in your browser

4. Add your RAWG API key in the API file if required
